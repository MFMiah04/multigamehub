# MultiGame Hub

A real-time multiplayer party game platform playable online or together in a room.

**Live:** https://multigame-hub.web.app

## Games
- **Spyfall** - One player is the spy. Everyone else knows the location. Can you figure out who doesn't belong?
- **Herd Mentality** - Answer questions the same way as the group. Think like the herd or get left behind.
- **Fakin' It** - One player gets a different prompt. Everyone else tries to spot who's faking it.
- **Imposter** - Find the imposter before they blend in.

## Tech Stack
JavaScript, Firebase Realtime Database, Firebase Hosting

## Running Locally
Open `index.html` in a browser or use a local server:
```bash
npx serve .
```
